## Definitions
| Glyph | Name     | Effect               | Usage        |
|-------|----------|----------------------|--------------|
| `⟷`   | `SWAP`   | `A B -- B A`         | `1 2 ⟷`      |
| `⍉`   | `ROT`    | `A B C -- B C A`     | `1 2 3 ⍉`    |
| `⟷2`  | `SWAP2`  | `A B C D -- C D A B` | `1 2 3 4 ⟷2` |
| `≍`   | `OVER`   | `A B -- A B A`       | `1 2 ≍`      |
| `:2`  | `DUP2`   | `A B -- A B A B`     | `1 2 :2`     |
| `/2`  | `SLASH2` | `A B -- `            | `1 2 /2`     |
